# BASIC and PRO Editions

ITsMagic has two editions on Google Play: **BASIC** and **PRO**. Both are made for creating games directly on Android, but they target different workflows.

## Download links

- **ITsMagic BASIC**: https://play.google.com/store/apps/details?id=com.itsmagic.enginestable
- **ITsMagic PRO**: https://play.google.com/store/apps/details?id=com.itsmagic.engine2

## Which edition should I install?

Use **BASIC** if you want to start quickly, learn the engine workflow, test ideas, and build projects with the core editor tools.

Use **PRO** if you need the full production workflow and advanced visual tools, including the features that are only unlocked in the PRO edition.

## Main differences

| Feature | BASIC | PRO |
| --- | --- | --- |
| Import models | Yes | Yes |
| Import textures | Yes | Yes |
| Scripts in Java, Lua, or NoCode | Yes | Yes |
| Lite graphics | Yes | Yes |
| Advanced graphics | No | Yes |
| Post-processing | No | Yes |
| VFX | No | Yes |
| APK export | No | Yes |
| Visual shader editor and Material Graph | No | Yes |

The PRO edition is the best fit for the advanced workflow shown on the landing page: stronger graphics, post-processing, VFX, visual shader editing, Material Graph work, and production-oriented exports.

## Updates

The **PRO** edition receives updates first. The **BASIC** edition receives those updates **6 months after the PRO edition**.

If you want the newest tools, fixes, and production features as soon as they are available, install **PRO**. If you prefer the free/basic workflow and can wait for delayed updates, install **BASIC**.
