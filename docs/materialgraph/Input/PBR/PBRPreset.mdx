# PBR Preset

**Menu:** `Input/PBR`

## Purpose

Outputs the texture set of the selected PBR preset so common surface maps can be wired quickly.

## Inputs

None.

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Albedo` | `TEXTURE` | - | - |
| `Normal` | `TEXTURE` | - | - |
| `Roughness` | `TEXTURE` | - | - |
| `Metallic` | `TEXTURE` | - | - |
| `AO` | `TEXTURE` | - | - |
| `Emissive` | `TEXTURE` | - | - |
| `Height` | `TEXTURE` | - | - |

## Notes

- Choose the preset in the node card to expose its texture set.
