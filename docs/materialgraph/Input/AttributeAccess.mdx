# Attribute

**Menu:** `Input`

## Purpose

Exposes a material attribute defined in graph settings as a runtime shader parameter.

## Inputs

None.

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Value` | `Selected attribute type` | - | - |

## Notes

- Choose the attribute in graph settings; the output type follows that attribute.
- `COLOR` attributes compile to `FLOAT4` in the shader.
