# Displacement

**Menu:** `Vertex`

## Purpose

Offsets vertex positions along a normal using the supplied height.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Position` | `FLOAT3` | - | Vertex stage. |
| `Normal` | `FLOAT3` | - | Vertex stage. |
| `Height` | `NUMBER` | - | Vertex stage. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Position` | `FLOAT3` | - | - |
