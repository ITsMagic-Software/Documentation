# Texture Sample

**Menu:** `Sampling`

## Purpose

Samples a 2D texture with the provided UVs and exposes both RGBA and individual channels.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Texture` | `TEXTURE` | - | Fragment stage. |
| `UV` | `FLOAT2` | - | Fragment stage. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
| `R` | `NUMBER` | - | - |
| `G` | `NUMBER` | - | - |
| `B` | `NUMBER` | - | - |
| `A` | `NUMBER` | - | - |
