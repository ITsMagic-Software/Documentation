# Cubemap Sample

**Menu:** `Sampling`

## Purpose

Samples a cubemap with a direction vector and exposes the resulting color and channels.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Cubemap` | `CUBEMAP` | - | Fragment stage. |
| `Direction` | `FLOAT2` | - | Fragment stage. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
| `R` | `NUMBER` | - | - |
| `G` | `NUMBER` | - | - |
| `B` | `NUMBER` | - | - |
| `A` | `NUMBER` | - | - |
