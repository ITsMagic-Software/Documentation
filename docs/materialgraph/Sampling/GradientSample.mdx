# Gradient Sample

**Menu:** `Sampling`

## Purpose

Evaluates a gradient resource at the supplied position and returns the sampled color.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Gradient` | `TEXTURE` | - | Fragment stage. |
| `X` | `NUMBER` | - | Fragment stage. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
