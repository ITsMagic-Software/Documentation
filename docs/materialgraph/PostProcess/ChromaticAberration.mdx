# Chromatic Aberration

**Menu:** `PostProcess`

## Purpose

Applies the `Chromatic Aberration` post-processing effect to the image.

| --- | --- | --- | --- |
| `Texture` | `TEXTURE` | - | - |
| `UV` | `FLOAT2` | - | - |
| `Center` | `FLOAT2` | - | - |
| `Falloff` | `NUMBER` | - | - |
| `Strength` | `NUMBER` | - | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
