# Cavity Ridge Valley

**Menu:** `PostProcess`

## Purpose

Applies the `Cavity Ridge Valley` post-processing effect to the image.

| --- | --- | --- | --- |
| `Texture` | `TEXTURE` | - | - |
| `UV` | `FLOAT2` | - | - |
| `Intensity` | `NUMBER` | - | - |
| `Ridge Strength` | `NUMBER` | - | - |
| `Valley Strength` | `NUMBER` | - | - |
| `Radius` | `NUMBER` | - | - |
| `Texel Size` | `FLOAT2` | - | - |
| `Ridge Color` | `FLOAT3` | - | - |
| `Valley Color` | `FLOAT3` | - | - |

## Outputs

| Name | Type | Default | Details |
