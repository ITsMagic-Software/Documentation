# Load Quaternion

**Menu:** `SaveGame`

## Purpose

Loads a Quaternion value.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Key` | `TEXT` | - | Save key. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Quaternion` | `QUATERNION` | - | Loaded value. |
