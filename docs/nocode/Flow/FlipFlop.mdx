# FlipFlop

**Menu:** `Flow`

## Purpose

Alternates between outputs `A` and `B` on each trigger and also exposes the chosen side through `Is A`.

## Inputs

None.

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `A` | `BRANCH` | - | - |
| `B` | `BRANCH` | - | - |
| `Is A` | `BOOLEAN` | - | - |
