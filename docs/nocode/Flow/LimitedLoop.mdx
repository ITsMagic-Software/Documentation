# Limited Loop

**Menu:** `Flow`

## Purpose

Runs `Loop Body` a fixed number of times, updates `Counter` every iteration, and then triggers `On Finish`.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Count` | `NUMBER` | `1` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Loop Body` | `BRANCH` | - | - |
| `Counter` | `NUMBER` | - | - |
| `On Finish` | `BRANCH` | - | - |
