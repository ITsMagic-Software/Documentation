# MultiGate

**Menu:** `Flow`

## Purpose

Sends each trigger to the next configured output in sequence and loops back to the first output after the last one.

## Inputs

None.

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Out 1..N` | `BRANCH` | - | Configurable amount of branch outputs. |

## Notes

- The number of branch outputs is configured by the node card field `Outputs`.
