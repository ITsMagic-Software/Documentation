# Branch

**Menu:** `Flow`

## Purpose

Evaluates `Condition` and routes execution to `On True` or `On False`.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Condition` | `BOOLEAN` | `false` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | - |
| `On False` | `BRANCH` | - | - |
