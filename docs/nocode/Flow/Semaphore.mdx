# Semaphore

**Menu:** `Flow`

## Purpose

Waits until all connected branch inputs have arrived before it releases the flow.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `In 1..N` | `BRANCH` | - | Configurable amount of branch inputs. |

## Outputs

None.

## Notes

- The number of branch inputs is configured by the node card field `Inputs`.
