# Loop Each

**Menu:** `Flow`

## Purpose

Iterates through every element of the input list, exposing both the current `Counter` and `Item` before triggering `On Finish`.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `List` | `LIST` | - | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Loop Body` | `BRANCH` | - | - |
| `Counter` | `NUMBER` | - | - |
| `Item` | `DYNAMIC` | - | - |
| `On Finish` | `BRANCH` | - | - |
