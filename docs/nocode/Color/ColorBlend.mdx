# Color Blend

**Menu:** `Color`

## Purpose

Interpolates between `Base` and `Blend` using `Opacity`, which is useful for soft color mixing.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Base` | `COLOR` | - | - |
| `Blend` | `COLOR` | - | - |
| `Opacity` | `NUMBER` | `0.5` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Out` | `COLOR` | - | - |
