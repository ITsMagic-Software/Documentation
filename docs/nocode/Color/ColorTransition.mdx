# Color Transition

**Menu:** `Color`

## Purpose

Animates a timed transition from `From` to `To`, updating `Out` until the blend is complete.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `From` | `COLOR` | - | - |
| `To` | `COLOR` | - | - |
| `Seconds` | `NUMBER` | `1.0` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Out` | `COLOR` | - | - |
