# Equal

**Menu:** `Compare`

## Purpose

Routes flow to `True` when `A` and `B` are equivalent.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `A` | `NUMBER` | - | - |
| `B` | `NUMBER` | - | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `True` | `BRANCH` | - | - |
| `False` | `BRANCH` | - | - |
