# Stop Animation

**Menu:** `Actions/Object/Animation`

## Purpose

Stops every animation whose name matches `Animation Name` across all `AnimationPlayer` components found on the target object.

When `Transition Delay` is greater than zero, the node fades the animation weight out with the same behavior used by `AnimationTrigger`.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | Target object used to search for `AnimationPlayer` components. |
| `Animation Name` | `TEXT` | - | Matches animation names with or without the `.anim` extension. |
| `Transition Delay` | `NUMBER` | `0.5` | Uses the same fade-out behavior as `AnimationTrigger`. |

## Outputs

None.
