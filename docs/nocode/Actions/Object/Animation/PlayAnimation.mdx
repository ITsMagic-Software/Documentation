# Play Animation

**Menu:** `Actions/Object/Animation`

## Purpose

Executes the `Play Animation` action inside the NoCode flow.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Animation Name` | `TEXT` | - | - |
| `Loop` | `BOOLEAN` | `true` | - |
| `Transition Delay` | `NUMBER` | `0.5` | - |

## Outputs

None.
