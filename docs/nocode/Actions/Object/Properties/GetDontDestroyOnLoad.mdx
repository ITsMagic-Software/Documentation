# Get Dont Destroy On Load

**Menu:** `Actions/Object/Properties`

## Purpose

Reads the `Dont Destroy On Load` property from the target object.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Dont Destroy On Load` | `BOOLEAN` | - | - |
