# Set Object Enabled

**Menu:** `Actions/Object/Properties`

## Purpose

Updates the `Object Enabled` property of the target object.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Enabled` | `BOOLEAN` | `true` | - |

## Outputs

None.
