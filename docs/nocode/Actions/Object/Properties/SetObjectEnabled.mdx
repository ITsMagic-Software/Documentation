# Set Object Enabled

**Menu:** `Actions/Object/Properties`

## Purpose

Updates `Object Enabled` with the values received from the inputs.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Enabled` | `BOOLEAN` | `true` | - |

## Outputs

None.
