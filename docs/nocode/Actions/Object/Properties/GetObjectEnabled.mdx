# Get Object Enabled

**Menu:** `Actions/Object/Properties`

## Purpose

Reads the `Object Enabled` property from the target object.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Enabled` | `BOOLEAN` | - | - |
