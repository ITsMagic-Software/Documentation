# Send Event

**Menu:** `Actions/Object/Events`

## Purpose

Dispatches a custom event to the target object without an extra payload.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | - | - |
| `Name` | `TEXT` | - | - |
| `To Hierarchy` | `BOOLEAN` | - | - |

## Outputs

None.
