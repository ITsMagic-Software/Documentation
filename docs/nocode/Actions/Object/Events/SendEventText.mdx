# Send Event Text

**Menu:** `Actions/Object/Events`

## Purpose

Dispatches a custom event to the target object with a text payload.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Name` | `TEXT` | - | - |
| `Value` | `TEXT` | - | - |
| `To Hierarchy` | `BOOLEAN` | `false` | - |

## Outputs

None.
