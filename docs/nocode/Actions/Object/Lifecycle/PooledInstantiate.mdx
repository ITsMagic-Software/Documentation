# Pooled Instantiate

**Menu:** `Actions/Object/Lifecycle`

## Purpose

Creates or reuses an object instance from the pool for repeated spawns.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object File` | `OBJECT_FILE` | - | - |
| `Pivot` | `GAME_OBJECT` | `Owner` | - |
| `Max Objects` | `NUMBER` | `10` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `GameObject` | `GAME_OBJECT` | - | - |
