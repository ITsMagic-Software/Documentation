# Smooth Displacement

**Menu:** `Actions/Object/Transform`

## Purpose

Displaces the owner over time by the requested offset, using physics components when available and transform translation otherwise.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `meters` | `FLOAT3` | - | - |
| `seconds` | `NUMBER` | `1.0` | - |

## Outputs

None.
