# Set Scale

**Menu:** `Actions/Object/Transform`

## Purpose

Updates `Scale` with the values received from the inputs.

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Scale` | `FLOAT3` | `float3(1, 1, 1)` | - |

## Outputs

None.
