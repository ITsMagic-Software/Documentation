# Set Global Position

**Menu:** `Actions/Object/Transform`

## Purpose

Sets `Global Position` on the target object transform.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Global Position` | `FLOAT3` | `float3(0, 0, 0)` | - |

## Outputs

None.
