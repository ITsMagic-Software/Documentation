# Set Global Position

**Menu:** `Actions/Object/Transform`

**Top flow input:** `Yes`

**Top flow output:** `Yes`

## Purpose

Updates `Global Position` with the values received from the inputs.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Global Position` | `FLOAT3` | `float3(0, 0, 0)` | - |

## Outputs

None.
