# Look To

**Menu:** `Actions/Object/Transform`

## Purpose

Rotates an object so it faces a target object, either instantly or over the requested number of seconds.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Target` | `GAME_OBJECT` | `Owner` | - |
| `Ignore Y` | `BOOLEAN` | `false` | - |
| `Seconds` | `NUMBER` | `0` | - |

## Outputs

None.
