# Limited Rotate

**Menu:** `Actions/Object/Transform`

## Purpose

Rotates the owner by the requested Euler angles over the specified duration instead of applying everything instantly.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `angles` | `FLOAT3` | - | - |
| `seconds` | `NUMBER` | `1.0` | - |

## Outputs

None.
