# Teleport To

**Menu:** `Actions/Object/Transform`

## Purpose

Moves the target object to a world position immediately or interpolates the travel when `Seconds` is greater than zero.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Position` | `FLOAT3` | `float3(0, 0, 0)` | - |
| `Seconds` | `NUMBER` | `0` | - |

## Outputs

None.
