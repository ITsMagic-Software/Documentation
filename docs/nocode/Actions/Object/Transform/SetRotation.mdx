# Set Rotation

**Menu:** `Actions/Object/Transform`

## Purpose

Updates `Rotation` with the values received from the inputs.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Rotation` | `QUATERNION` | `float4(0, 0, 0, 1)` | - |

## Outputs

None.
