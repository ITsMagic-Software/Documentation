# Set Global Scale

**Menu:** `Actions/Object/Transform`

## Purpose

Sets `Global Scale` on the target object transform.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Global Scale` | `FLOAT3` | `float3(1, 1, 1)` | - |

## Outputs

None.
