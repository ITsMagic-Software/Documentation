# Is Colliding With

**Menu:** `Actions/Object/Queries`

## Purpose

Checks whether `Colliding With` is true for the target object.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Other Name` | `TEXT` | - | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | - |
| `On False` | `BRANCH` | - | - |
