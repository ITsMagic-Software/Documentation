# Is In Front

**Menu:** `Actions/Object/Queries`

## Purpose

Checks whether `In Front` is true for the target object.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object A` | `GAME_OBJECT` | `Owner` | - |
| `Object B` | `GAME_OBJECT` | `Owner` | - |
| `Tolerance` | `NUMBER` | `30` | - |
| `Max Distance` | `NUMBER` | `1000` | Limits the maximum distance between `Object A` and `Object B` for the check to pass. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | - |
| `On False` | `BRANCH` | - | - |
