# Set Material Color

**Menu:** `Actions/Material/Colors`

## Purpose

Writes a new value to `Material Color` on the selected material.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Material` | `MATERIAL` | `Owner` | - |
| `Color` | `COLOR` | `float4(1, 1, 1, 1)` | - |

## Outputs

None.
