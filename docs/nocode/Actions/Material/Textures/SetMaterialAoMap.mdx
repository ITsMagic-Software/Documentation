# Set Material AO Map

**Menu:** `Actions/Material/Textures`

## Purpose

Updates `Material AO Map` with the values received from the inputs.

If the texture input uses a file default, it is loaded during world async loading before gameplay starts.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Material` | `MATERIAL` | - | - |

## Outputs

None.
