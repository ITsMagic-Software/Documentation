# Get Object

**Menu:** `Actions/Component/Properties`

## Purpose

Reads `Object` from the selected component.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Component` | `ANY_COMPONENT` | `Owner` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | - | - |
