# Get Enabled

**Menu:** `Actions/Component/Properties`

## Purpose

Reads `Enabled` from the selected component.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Component` | `ANY_COMPONENT` | `Owner` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Enabled` | `BOOLEAN` | - | - |
