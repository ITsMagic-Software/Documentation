# Pick Component

**Menu:** `Actions/Component/Queries`

## Purpose

Searches the supplied object for the component type selected in the node card.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `owner` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Found` | `BRANCH` | - | - |
| `Missing` | `BRANCH` | - | - |
| `Component` | `Selected component type` | - | - |

## Notes

- The `Component` output type depends on the component selected in the node card.
