# Component Method

**Menu:** `Actions/Component`

## Purpose

Calls a Java component method selected in the node card and creates inputs for the method parameters when needed.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Component` | `Selected component type` | `Pick first component` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Return` | `Method return type` | - | Only shown when the selected method returns a supported value. |

## Notes

- This node is configured from a Java component method.
- Additional inputs are created from the selected method parameters.
- The output only exists when the selected method returns a supported value.
