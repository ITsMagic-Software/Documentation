# Trace Ray (O)

**Menu:** `Actions/Ray`

## Purpose

Casts a ray starting from the chosen object and exposes hit branches plus detailed collision data.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | - | - |
| `Direction` | `DIRECTION` | - | - |
| `Distance` | `NUMBER` | - | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `On Hit` | `BRANCH` | - | - |
| `No Hit` | `BRANCH` | - | - |
| `Object` | `GAME_OBJECT` | - | - |
| `Collider Object` | `GAME_OBJECT` | - | - |
| `Collider` | `COMPONENT_COLLIDER` | - | - |
| `Point` | `FLOAT3` | - | - |
| `Face Normal` | `FLOAT3` | - | - |
| `Normal` | `FLOAT3` | - | - |
| `Distance` | `NUMBER` | - | - |
| `UV Coord` | `FLOAT2` | - | - |
