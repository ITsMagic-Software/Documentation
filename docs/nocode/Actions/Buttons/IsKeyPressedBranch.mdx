# Is Key Pressed

**Menu:** `Actions/Buttons`

## Purpose

Checks the condition `Key Pressed` and routes the flow based on the result.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Key Name` | `TEXT` | - | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `True` | `BRANCH` | - | - |
| `False` | `BRANCH` | - | - |
