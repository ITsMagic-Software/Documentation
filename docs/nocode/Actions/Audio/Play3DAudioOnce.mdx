# Play 3D Audio Once

**Menu:** `Actions/Audio`

## Purpose

Plays a 3D sound a single time in the current context.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Sound File` | `SOUND_FILE` | - | - |
| `Diameter` | `NUMBER` | `20` | - |
| `Volume` | `NUMBER` | `1` | - |
| `Pitch` | `NUMBER` | `1` | - |
| `Speed` | `NUMBER` | `1` | - |
| `Load Mode` | `SOUND_LOAD_MODE` | - | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `audio object` | `GAME_OBJECT` | - | - |
