# Is Empty

**Menu:** `Actions/List`

## Purpose

Checks whether the input list is empty.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `List` | `LIST` | - | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | Fired when the list is empty. |
| `On False` | `BRANCH` | - | Fired when the list is not empty. |
