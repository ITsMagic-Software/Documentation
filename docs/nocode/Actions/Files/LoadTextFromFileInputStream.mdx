# Load Text From File (InputStream)

**Menu:** `Files`

## Purpose

Loads text from an `InputStream` and returns the resulting text.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Input Stream` | `INPUT_STREAM` | - | Accepts an `InputStream` value. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | Loaded text. |
