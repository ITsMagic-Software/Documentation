# Load Text From File (Project File)

**Menu:** `Files`

## Purpose

Loads text from a selected `ProjectFile` and returns the resulting text.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Project File` | `PROJECT_FILE` | - | Accepts a `ProjectFile` value. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | Loaded text. |
