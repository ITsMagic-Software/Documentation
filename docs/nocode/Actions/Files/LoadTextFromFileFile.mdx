# Load Text From File (File)

**Menu:** `Files`

## Purpose

Loads text from a selected `File` and returns the resulting text.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `File` | `FILE` | - | Accepts a `File` value. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | Loaded text. |
