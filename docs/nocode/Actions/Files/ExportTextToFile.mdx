# Export Text To File

**Menu:** `Files`

## Purpose

Exports text to the file path provided in the `File Path` input.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | Text content to write. |
| `File Path` | `TEXT` | - | File path to save to. |

## Outputs

None.
