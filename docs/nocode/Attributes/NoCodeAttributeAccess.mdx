# Get Attribute

**Menu:** `Attributes`

## Purpose

Reads the value related to `Attribute` and exposes the result on the node outputs.

## Inputs

None.

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Value` | `Selected attribute type` | - | - |

## Notes

- The `Value` output type depends on the attribute selected in the node card.
