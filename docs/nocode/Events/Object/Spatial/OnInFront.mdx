# On In Front

**Menu:** `Events/Object/Spatial`

## Purpose

Fires when `On In Front` happens.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object A` | `GAME_OBJECT` | `Owner` | - |
| `Object B` | `GAME_OBJECT` | `Owner` | - |
| `Tolerance` | `NUMBER` | `30` | - |
| `Max Distance` | `NUMBER` | `1000` | Limits the maximum distance between `Object A` and `Object B` for the event to fire. |

## Outputs

None.
