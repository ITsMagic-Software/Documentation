# On Grounded Begin

**Menu:** `Events/Object/Movement`

## Purpose

Fires once when the target object enters the grounded state.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | Target object that should contain a `Characterbody`. |

## Outputs

None.

## Notes

- If the target object does not have a `Characterbody`, the node behaves as `false`.
