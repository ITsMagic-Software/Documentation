# Is Grounded

**Menu:** `Events/Object/Movement`

## Purpose

Checks the grounded state of the target object every frame and routes execution through `On True` or `On False`.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | Target object that should contain a `Characterbody`. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | Fired while the target is grounded. |
| `On False` | `BRANCH` | - | Fired while the target is not grounded. |

## Notes

- If the target object does not have a `Characterbody`, the node behaves as `false`.
