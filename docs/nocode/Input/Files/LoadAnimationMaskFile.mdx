# Load Animation Mask File

**Menu:** `Input/Files`

## Purpose

Loads an animation mask from an `AnimationMaskFile` and returns the resulting mask instance.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Animation Mask File` | `ANIMATION_MASK_FILE` | - | Accepts an `AnimationMaskFile` value. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Animation Mask` | `ANIMATION_MASK` | - | Loaded animation mask instance. |
