# Load Animation File

**Menu:** `Input/Files`

## Purpose

Loads an animation from an `AnimationFile` and returns the resulting animation instance.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Animation File` | `ANIMATION_FILE` | - | Accepts an `AnimationFile` value. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Animation` | `ANIMATION` | - | Loaded animation instance. |
