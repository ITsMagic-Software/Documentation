# Inverse Lerp

**Menu:** `Math/Interpolation`

## Purpose

Converts a value into its normalized position between two limits.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `A` | `NUMBER` | `0.0` | - |
| `B` | `NUMBER` | `1.0` | - |
| `Value` | `NUMBER` | - | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `T` | `NUMBER` | - | - |
