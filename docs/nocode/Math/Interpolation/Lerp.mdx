# Lerp

**Menu:** `Math/Interpolation`

## Purpose

Interpolates between `A` and `B` using a blend factor.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `A` | `DYNAMIC` | - | - |
| `B` | `DYNAMIC` | - | - |
| `T` | `NUMBER` | `0.5` | - |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Out` | `DYNAMIC` | - | - |
