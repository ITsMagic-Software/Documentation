# Triangle Normal

**Menu:** `Math/Vector`

## Purpose

Calculates the normalized triangle normal from three `FLOAT3` vertices using the `A`, `B`, `C` winding order.

## Inputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `A` | `FLOAT3` | - | First vertex. |
| `B` | `FLOAT3` | - | Second vertex. |
| `C` | `FLOAT3` | - | Third vertex. |

## Outputs

| Name | Type | Default | Details |
| --- | --- | --- | --- |
| `Out` | `FLOAT3` | - | Normalized triangle normal. |
