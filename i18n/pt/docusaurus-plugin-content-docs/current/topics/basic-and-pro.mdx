# Edições BASIC e PRO

A ITsMagic possui duas edições na Google Play: **BASIC** e **PRO**. As duas são feitas para criar jogos diretamente no Android, mas atendem fluxos de trabalho diferentes.

## Links de download

- **ITsMagic BASIC**: https://play.google.com/store/apps/details?id=com.itsmagic.enginestable
- **ITsMagic PRO**: https://play.google.com/store/apps/details?id=com.itsmagic.engine2

## Qual edição devo instalar?

Use a **BASIC** se você quer começar rápido, aprender o fluxo da engine, testar ideias e criar projetos com as ferramentas principais do editor.

Use a **PRO** se você precisa do fluxo completo de produção e das ferramentas visuais avançadas, incluindo os recursos liberados apenas na edição PRO.

## Principais diferenças

| Recurso | BASIC | PRO |
| --- | --- | --- |
| Importar modelos | Sim | Sim |
| Importar texturas | Sim | Sim |
| Scripts em Java, Lua ou NoCode | Sim | Sim |
| Gráficos lite | Sim | Sim |
| Gráficos avançados | Não | Sim |
| Pós-processamento | Não | Sim |
| VFX | Não | Sim |
| Exportação APK | Não | Sim |
| Editor visual de shaders e Material Graph | Não | Sim |

A edição PRO é a opção mais indicada para o fluxo avançado mostrado na landing page: gráficos mais fortes, pós-processamento, VFX, edição visual de shaders, trabalho com Material Graph e exportação voltada para produção.

## Atualizações

A edição **PRO** recebe atualizações primeiro. A edição **BASIC** recebe essas atualizações **6 meses depois da edição PRO**.

Se você quer as ferramentas, correções e recursos de produção mais recentes assim que eles estiverem disponíveis, instale a **PRO**. Se você prefere o fluxo básico/gratuito e pode aguardar as atualizações atrasadas, instale a **BASIC**.
