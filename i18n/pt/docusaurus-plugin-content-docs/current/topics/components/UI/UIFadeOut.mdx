# UIFadeOut

Efeito de fade out exposto pelo runtime Java.

## Criar no Editor

1. Garanta que exista um `UIController` na cena.
2. Crie um GameObject e adicione `UIRect`.
3. Adicione o componente `UIFadeOut`.

## Como usar

1. Obtenha o componente com `myObject.findComponent(UIFadeOut.class)`.
2. Leia ou altere os valores expostos a partir do Java.
3. Teste o comportamento em Play mode.

## Exemplo em Java

```java
SpatialObject myObject = /* seu objeto */;
UIFadeOut comp = myObject.findComponent(UIFadeOut.class);
if (comp != null) {
    // define o valor de opacity:
    comp.opacity = 1f;
    // l? o valor de opacity:
    float opacityValue = comp.opacity;

}
```
