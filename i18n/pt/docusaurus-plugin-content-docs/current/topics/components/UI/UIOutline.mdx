# UIOutline

Efeito de contorno exposto pelo runtime Java.

## Criar no Editor

1. Garanta que exista um `UIController` na cena.
2. Crie um GameObject e adicione `UIRect`.
3. Adicione o componente `UIOutline`.

## Como usar

1. Obtenha o componente com `myObject.findComponent(UIOutline.class)`.
2. Leia ou altere os valores expostos a partir do Java.
3. Teste o comportamento em Play mode.

## Exemplo em Java

```java
SpatialObject myObject = /* seu objeto */;
UIOutline comp = myObject.findComponent(UIOutline.class);
if (comp != null) {
    // define o valor de outlineColor:
    comp.outlineColor = new Color(255, 255, 255, 255);
    // l? o valor de outlineColor:
    Color outlineColorValue = comp.outlineColor;

    // define o valor de thickness:
    comp.thickness = 1f;
    // l? o valor de thickness:
    float thicknessValue = comp.thickness;

    // define o valor de threshold:
    comp.threshold = 1f;
    // l? o valor de threshold:
    float thresholdValue = comp.threshold;

}
```
