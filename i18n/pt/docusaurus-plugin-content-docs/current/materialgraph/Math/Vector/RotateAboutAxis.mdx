# Rotate About Axis

**Menu:** `Math/Vector`

## Funcao

Rotaciona um vetor em torno do eixo informado.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Vector` | `FLOAT3` | - | - |
| `Axis` | `FLOAT3` | - | - |
| `Angle` | `NUMBER` | - | - |
