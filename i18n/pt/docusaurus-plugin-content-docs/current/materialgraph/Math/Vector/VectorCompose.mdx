# Vector Compose

**Menu:** `Math/Vector`

## Funcao

Combina componentes escalares em um vetor para reutilizacao no shader.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `X` | `NUMBER` | - | - |
| `Y` | `NUMBER` | - | - |
| `Z` | `NUMBER` | - | - |
| `W` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Out` | `FLOAT4` | - | - |
