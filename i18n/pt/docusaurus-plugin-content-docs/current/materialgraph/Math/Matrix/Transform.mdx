# Transform

**Menu:** `Math/Matrix`

## Funcao

Monta uma transformacao com posicao, rotacao e escala a partir das entradas e a aplica ao vetor recebido.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Vector` | `FLOAT3` | - | - |
| `Position` | `FLOAT3` | - | - |
| `Rotation` | `FLOAT3` | - | - |
| `Scale` | `FLOAT3` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Out` | `FLOAT3` | - | - |
