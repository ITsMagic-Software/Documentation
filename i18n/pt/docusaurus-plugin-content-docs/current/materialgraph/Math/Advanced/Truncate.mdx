# Truncate

**Menu:** `Math/Advanced`

## Funcao

Remove a parte fracionaria de `In` sem arredondar.

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `In` | `DYNAMIC` | - | - |
