# Inverse Lerp

**Menu:** `Math/Interpolation`

## Funcao

Converte um valor para sua posicao normalizada entre dois limites.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `A` | `NUMBER` | - | - |
| `B` | `NUMBER` | - | - |
| `Value` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `T` | `NUMBER` | - | - |
