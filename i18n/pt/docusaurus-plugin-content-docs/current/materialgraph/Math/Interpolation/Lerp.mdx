# Lerp

**Menu:** `Math/Interpolation`

## Funcao

Interpola entre `A` e `B` usando um fator de mistura.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `A` | `DYNAMIC` | - | - |
| `B` | `DYNAMIC` | - | - |
| `T` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Out` | `DYNAMIC` | - | - |
