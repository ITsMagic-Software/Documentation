# Map Range

**Menu:** `Math/Range`

## Funcao

Mapeia um valor de uma faixa para outra.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Value` | `DYNAMIC` | `0` | - |
| `From Min` | `DYNAMIC` | `0` | - |
| `From Max` | `DYNAMIC` | `1` | - |
| `To Min` | `DYNAMIC` | `0` | - |
| `To Max` | `DYNAMIC` | `1` | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Out` | `DYNAMIC` | - | - |
