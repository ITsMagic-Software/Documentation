# Fresnel

**Menu:** `Math`

## Funcao

Calcula um fator Fresnel a partir da direcao de visao e da normal.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Normal` | `FLOAT3` | - | Etapa de fragment. |
| `View` | `FLOAT3` | - | Etapa de fragment. |
| `Power` | `NUMBER` | - | Etapa de fragment. |
