# Normal Intensity

**Menu:** `Vector/Normal`

## Funcao

Processa vetores normais com `Normal Intensity` para controlar o detalhe de iluminacao da superficie.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Normal` | `FLOAT3` | - | - |
| `Intensity` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Out` | `FLOAT3` | - | - |
