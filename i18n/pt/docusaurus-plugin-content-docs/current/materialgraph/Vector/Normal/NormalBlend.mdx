# Normal Blend

**Menu:** `Vector/Normal`

## Funcao

Processa vetores normais com `Normal Blend` para controlar o detalhe de iluminacao da superficie.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `A` | `FLOAT3` | - | - |
| `B` | `FLOAT3` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Out` | `FLOAT3` | - | - |
