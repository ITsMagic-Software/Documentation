# Normal Strength

**Menu:** `Vector/Normal`

## Funcao

Processa vetores normais com `Normal Strength` para controlar o detalhe de iluminacao da superficie.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Normal` | `FLOAT3` | - | - |
| `Strength` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Out` | `FLOAT3` | - | - |
