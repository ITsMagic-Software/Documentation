# Outline

**Menu:** `PostProcess`

## Funcao

Aplica o efeito de pos-processamento `Outline` sobre a imagem.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Texture` | `TEXTURE` | - | - |
| `UV` | `FLOAT2` | - | - |
| `Line Color` | `FLOAT4` | - | - |
| `Thickness` | `NUMBER` | - | - |
| `Threshold` | `NUMBER` | - | - |
| `Texel Size` | `FLOAT2` | - | - |
