# Cavity Ridge Valley

**Menu:** `PostProcess`

## Funcao

Aplica o efeito de pos-processamento `Cavity Ridge Valley` sobre a imagem.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Texture` | `TEXTURE` | - | - |
| `UV` | `FLOAT2` | - | - |
| `Intensity` | `NUMBER` | - | - |
| `Ridge Strength` | `NUMBER` | - | - |
| `Valley Strength` | `NUMBER` | - | - |
| `Radius` | `NUMBER` | - | - |
| `Texel Size` | `FLOAT2` | - | - |
| `Ridge Color` | `FLOAT3` | - | - |
| `Valley Color` | `FLOAT3` | - | - |
