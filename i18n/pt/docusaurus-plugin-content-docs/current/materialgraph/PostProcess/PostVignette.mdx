# Vignette

**Menu:** `PostProcess`

## Funcao

Aplica o efeito de pos-processamento `Vignette` sobre a imagem.

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
| `UV` | `FLOAT2` | - | - |
| `Mid Point` | `NUMBER` | - | - |
| `Roundness` | `NUMBER` | - | - |
| `Feather` | `NUMBER` | - | - |
| `Vignette Color` | `FLOAT4` | - | - |
| `Aspect` | `NUMBER` | - | - |
