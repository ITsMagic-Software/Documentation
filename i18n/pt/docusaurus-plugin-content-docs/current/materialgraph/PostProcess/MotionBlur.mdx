# Motion Blur

**Menu:** `PostProcess`

## Funcao

Aplica o efeito de pos-processamento `Motion Blur` sobre a imagem.

| `History Color` | `FLOAT4` | - | - |
| `Alpha` | `NUMBER` | - | - |
| `Reset` | `NUMBER` | - | - |
