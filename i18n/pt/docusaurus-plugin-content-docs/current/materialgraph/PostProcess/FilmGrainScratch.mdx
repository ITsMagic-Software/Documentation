# Film Grain Scratch

**Menu:** `PostProcess`

## Funcao

Aplica o efeito de pos-processamento `Film Grain Scratch` sobre a imagem.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
| `UV` | `FLOAT2` | - | - |
| `Intensity` | `NUMBER` | - | - |
| `Grain Strength` | `NUMBER` | - | - |
| `Grain Size` | `NUMBER` | - | - |
| `Scratch Strength` | `NUMBER` | - | - |
| `Scratch Density` | `NUMBER` | - | - |
| `Scratch Width` | `NUMBER` | - | - |
| `Flicker Strength` | `NUMBER` | - | - |
| `Time` | `NUMBER` | - | - |
