# Chromatic Aberration

**Menu:** `PostProcess`

## Funcao

Aplica o efeito de pos-processamento `Chromatic Aberration` sobre a imagem.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Texture` | `TEXTURE` | - | - |
| `UV` | `FLOAT2` | - | - |
| `Center` | `FLOAT2` | - | - |
| `Falloff` | `NUMBER` | - | - |
| `Strength` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
