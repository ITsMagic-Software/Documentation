# Night Vision

**Menu:** `PostProcess`

## Funcao

Aplica o efeito de pos-processamento `Night Vision` sobre a imagem.

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
| `UV` | `FLOAT2` | - | - |
| `Intensity` | `NUMBER` | - | - |
| `Brightness` | `NUMBER` | - | - |
| `Contrast` | `NUMBER` | - | - |
| `Noise Strength` | `NUMBER` | - | - |
| `Scanline Strength` | `NUMBER` | - | - |
| `Scanline Count` | `NUMBER` | - | - |
| `Vignette Strength` | `NUMBER` | - | - |
| `Aspect` | `NUMBER` | - | - |
| `Tint Color` | `FLOAT3` | - | - |
| `Time` | `NUMBER` | - | - |
