# Perlin Noise

**Menu:** `Noise`

## Funcao

Gera o padrao procedural `Perlin Noise` para mascaras, distorcoes ou variacao de superficie.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `UV` | `FLOAT2` | - | - |
| `Scale` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Value` | `NUMBER` | - | - |


## Alta Precisao

- Os calculos procedurais internos usam operacoes numericas de alta precisao para preservar resultados estaveis quando as coordenadas sao grandes ou compartilhadas entre as etapas de vertex e fragment.
