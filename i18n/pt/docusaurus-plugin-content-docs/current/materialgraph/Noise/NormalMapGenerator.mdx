# NormalMap Generator

**Menu:** `Noise`

## Funcao

Gera o padrao procedural `NormalMap Generator` para mascaras, distorcoes ou variacao de superficie.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Texture` | `TEXTURE` | - | - |
| `UV` | `FLOAT2` | - | - |
| `Bias` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Normal` | `FLOAT3` | - | - |
