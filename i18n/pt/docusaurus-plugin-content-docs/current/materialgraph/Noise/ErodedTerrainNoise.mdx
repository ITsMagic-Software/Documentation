# Eroded Terrain Noise

**Menu:** `Noise`

## Funcao

Gera o padrao procedural `Eroded Terrain Noise` para mascaras, distorcoes ou variacao de superficie.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `UV` | `FLOAT2` | - | - |
| `Scale` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Height/Normal` | `FLOAT3` | - | - |
