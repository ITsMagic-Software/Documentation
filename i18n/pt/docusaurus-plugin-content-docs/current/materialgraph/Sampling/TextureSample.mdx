# Texture Sample

**Menu:** `Sampling`

## Funcao

Faz a amostragem de uma textura 2D com os UVs informados e expoe RGBA e canais individuais.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Texture` | `TEXTURE` | - | Etapa de fragment. |
| `UV` | `FLOAT2` | - | Etapa de fragment. |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
| `R` | `NUMBER` | - | - |
| `G` | `NUMBER` | - | - |
| `B` | `NUMBER` | - | - |
| `A` | `NUMBER` | - | - |
