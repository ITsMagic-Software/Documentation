# Gradient Sample

**Menu:** `Sampling`

## Funcao

Avalia um recurso de gradiente na posicao informada e retorna a cor amostrada.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Gradient` | `TEXTURE` | - | Etapa de fragment. |
| `X` | `NUMBER` | - | Etapa de fragment. |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
