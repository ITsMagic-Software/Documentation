# Cubemap Sample

**Menu:** `Sampling`

## Funcao

Faz a amostragem de um cubemap usando um vetor de direcao e expoe a cor resultante e seus canais.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Cubemap` | `CUBEMAP` | - | Etapa de fragment. |
| `Direction` | `FLOAT2` | - | Etapa de fragment. |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Color` | `FLOAT4` | - | - |
| `R` | `NUMBER` | - | - |
| `G` | `NUMBER` | - | - |
| `B` | `NUMBER` | - | - |
| `A` | `NUMBER` | - | - |
