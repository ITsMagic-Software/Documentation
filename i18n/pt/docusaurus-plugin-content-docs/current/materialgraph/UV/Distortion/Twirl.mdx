# Twirl

**Menu:** `UV/Distortion`

## Funcao

Modifica coordenadas UV com `Twirl` para controlar repeticao, rotacao ou distorcao.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `UV` | `FLOAT2` | - | - |
| `Center` | `FLOAT2` | - | - |
| `Strength` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Out` | `FLOAT2` | - | - |
