# Rotate

**Menu:** `UV`

## Funcao

Modifica coordenadas UV com `Rotate` para controlar repeticao, rotacao ou distorcao.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `UV` | `FLOAT2` | - | - |
| `Angle` | `NUMBER` | - | - |
| `Center` | `FLOAT2` | - | - |
