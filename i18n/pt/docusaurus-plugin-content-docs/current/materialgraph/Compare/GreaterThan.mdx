# Greater Than

**Menu:** `Compare`

## Funcao

Produz uma condicao verdadeira quando `A` e maior que `B`.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `A` | `NUMBER` | - | - |
| `B` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `True` | `BRANCH` | - | - |
| `False` | `BRANCH` | - | - |
