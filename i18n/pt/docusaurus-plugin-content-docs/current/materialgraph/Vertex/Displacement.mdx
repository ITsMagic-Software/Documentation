# Displacement

**Menu:** `Vertex`

## Funcao

Desloca vertices ao longo da normal usando a altura fornecida.

## Entradas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Position` | `FLOAT3` | - | Etapa de vertex. |
| `Normal` | `FLOAT3` | - | Etapa de vertex. |
| `Height` | `NUMBER` | - | Etapa de vertex. |
