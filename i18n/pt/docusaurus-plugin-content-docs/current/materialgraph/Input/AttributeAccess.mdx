# Attribute

**Menu:** `Input`

## Funcao

Expoe como parametro de shader um atributo de material definido nas configuracoes do grafo.

## Entradas

Nenhuma.

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Value` | `Selected attribute type` | - | - |

## Observacoes

- Escolha o atributo nas configuracoes do grafo; o tipo da saida acompanha esse atributo.
- Atributos `COLOR` compilam como `FLOAT4` no shader.
