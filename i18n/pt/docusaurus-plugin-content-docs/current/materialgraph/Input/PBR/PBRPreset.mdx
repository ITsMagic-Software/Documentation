# PBR Preset

**Menu:** `Input/PBR`

## Funcao

Expoe o conjunto de texturas do preset PBR selecionado para ligar rapidamente mapas comuns de superficie.

## Entradas

Nenhuma.

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Albedo` | `TEXTURE` | - | - |
| `Normal` | `TEXTURE` | - | - |
| `Roughness` | `TEXTURE` | - | - |
| `Metallic` | `TEXTURE` | - | - |
| `AO` | `TEXTURE` | - | - |
| `Emissive` | `TEXTURE` | - | - |
| `Height` | `TEXTURE` | - | - |

## Observacoes

- Escolha o preset no card do node para expor seu conjunto de texturas.
