# Normalized Time

**Menu:** `Input/Time`

## Funcao

Expoe um valor de tempo normalizado na faixa `0..1`. O node acumula o tempo no Java e envia o float pronto para o shader.

## Propriedades

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Speed` | `NUMBER` | `1.0` | Multiplica `deltaTime` antes de avancar o timer normalizado. |

## Saidas

| Nome | Tipo | Default | Detalhes |
| --- | --- | --- | --- |
| `Normalized Time` | `NUMBER` | `0..1` | Valor de tempo normalizado pronto para uso. |
