# Subtract

**Menu:** `Math/Basic`

## Funcao

Aplica a operacao matematica `Subtract` sobre os valores conectados.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `A` | `DYNAMIC` | - | - |
| `B` | `DYNAMIC` | - | - |

## Nota do editor

Quando a entrada `B` estiver desconectada, o editor agora exibe um campo numerico de valor padrao mesmo com o slot sendo `DYNAMIC`.
