# Number to Text

**Menu:** `Math/Basic`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Nao`

## Funcao

Aplica a operacao matematica `Number to Text` sobre os valores conectados.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Number` | `NUMBER` | `0.0` | - |
| `Decimals` | `NUMBER` | `2.0` | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | - |
