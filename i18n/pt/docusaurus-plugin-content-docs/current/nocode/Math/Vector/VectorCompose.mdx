# Vector Compose

**Menu:** `Math/Vector`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Nao`

## Funcao

Aplica a operacao matematica `Vector Compose` sobre os valores conectados.

| Nome | Tipo | Padrao | Detalhes |

| `Y` | `NUMBER` | - | - |
| `Z` | `NUMBER` | - | - |
| `W` | `NUMBER` | - | - |
