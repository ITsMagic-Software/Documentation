# Triangle Normal

**Menu:** `Math/Vector`

## Funcao

Calcula a normal normalizada do triangulo a partir de tres vertices `FLOAT3` usando a ordem `A`, `B`, `C`.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `A` | `FLOAT3` | - | Primeiro vertice. |
| `B` | `FLOAT3` | - | Segundo vertice. |
| `C` | `FLOAT3` | - | Terceiro vertice. |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Out` | `FLOAT3` | - | Normal do triangulo normalizada. |
