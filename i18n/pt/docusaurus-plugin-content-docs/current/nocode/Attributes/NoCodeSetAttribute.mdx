# Set Attribute

**Menu:** `Attributes`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Sim`

## Funcao

Atualiza `Attribute` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Value` | `Selected attribute type` | - | - |

## Saidas

Nenhuma.

## Observacoes

- O tipo da entrada `Value` depende do atributo selecionado no card do node.
