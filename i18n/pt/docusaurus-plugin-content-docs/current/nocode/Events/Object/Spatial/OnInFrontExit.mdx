# On In Front Exit

**Menu:** `Events/Object/Spatial`

## Funcao

Dispara quando `On In Front Exit` acontece.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object A` | `GAME_OBJECT` | `Owner` | - |
| `Object B` | `GAME_OBJECT` | `Owner` | - |
| `Tolerance` | `NUMBER` | `30` | - |
| `Max Distance` | `NUMBER` | `1000` | Limita a distancia maxima entre `Object A` e `Object B` para o evento disparar. |

## Saidas

Nenhuma.
