# On Key Up

**Menu:** `Events/Buttons`

## Funcao

Evento que dispara quando a condicao `On Key Up` acontece.

| Nome | Tipo | Padrao | Detalhes |

| `Key Name` | `TEXT` | `keyname` | - |
