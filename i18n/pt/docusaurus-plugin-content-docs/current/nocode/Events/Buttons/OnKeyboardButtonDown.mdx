# On Keyboard Button Down

**Menu:** `Events/Buttons`

## Funcao

Evento que dispara quando a condicao `On Keyboard Button Down` acontece.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Button Name` | `TEXT` | `buttonname` | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `True` | `BRANCH` | - | - |
| `False` | `BRANCH` | - | - |
