# Pick Component

**Menu:** `Actions/Component/Queries`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Nao`

## Funcao

Executa a acao `Pick Component` dentro do fluxo NoCode.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `owner` | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Found` | `BRANCH` | - | - |
| `Missing` | `BRANCH` | - | - |
| `Component` | `Selected component type` | - | - |

## Observacoes

- O tipo da saida `Component` depende do componente selecionado no card do node.
