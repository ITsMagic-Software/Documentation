# Set Enabled

**Menu:** `Actions/Component/Properties`

## Funcao

Atualiza `Enabled` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Component` | `ANY_COMPONENT` | `Owner` | - |
| `Enabled` | `BOOLEAN` | `true` | - |

## Saidas

Nenhuma.
