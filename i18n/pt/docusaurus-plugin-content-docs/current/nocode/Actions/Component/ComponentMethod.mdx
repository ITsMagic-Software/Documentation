# Component Method

**Menu:** `Actions/Component`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Sim`

## Funcao

Executa a acao `Component Method` dentro do fluxo NoCode.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Component` | `Selected component type` | `Pick first component` | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Return` | `Method return type` | - | Only shown when the selected method returns a supported value. |

## Observacoes

- Este node e configurado a partir de um metodo de componente Java.
- Entradas adicionais sao criadas a partir dos parametros do metodo selecionado.
- A saida so existe quando o metodo selecionado retorna um valor suportado.
