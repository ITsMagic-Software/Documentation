# Is Colliding With

**Menu:** `Actions/Object/Queries`

## Funcao

Verifica a condicao `Colliding With` e encaminha o fluxo conforme o resultado.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Other Name` | `TEXT` | - | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | - |
| `On False` | `BRANCH` | - | - |
