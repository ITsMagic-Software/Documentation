# Is 3D Touched

**Menu:** `Actions/Object/Queries`

## Funcao

Verifica a condicao `3D Touched` e encaminha o fluxo conforme o resultado.

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |

| `On False` | `BRANCH` | - | - |
