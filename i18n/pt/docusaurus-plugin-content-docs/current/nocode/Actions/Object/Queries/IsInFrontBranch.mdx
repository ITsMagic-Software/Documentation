# Is In Front

**Menu:** `Actions/Object/Queries`

## Funcao

Verifica se `In Front` e verdadeiro para o objeto alvo.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object A` | `GAME_OBJECT` | `Owner` | - |
| `Object B` | `GAME_OBJECT` | `Owner` | - |
| `Tolerance` | `NUMBER` | `30` | - |
| `Max Distance` | `NUMBER` | `1000` | Limita a distancia maxima entre `Object A` e `Object B` para a checagem passar. |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | - |
| `On False` | `BRANCH` | - | - |
