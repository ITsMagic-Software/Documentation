# Is 3D Aimed

**Menu:** `Actions/Object/Queries`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Nao`

## Funcao

Verifica a condicao `3D Aimed` e encaminha o fluxo conforme o resultado.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Max Distance` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | - |
| `On False` | `BRANCH` | - | - |
