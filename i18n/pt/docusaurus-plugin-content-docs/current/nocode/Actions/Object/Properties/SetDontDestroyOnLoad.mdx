# Set Dont Destroy On Load

**Menu:** `Actions/Object/Properties`

## Funcao

Atualiza `Dont Destroy On Load` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Dont Destroy On Load` | `BOOLEAN` | `false` | - |

## Saidas

Nenhuma.
