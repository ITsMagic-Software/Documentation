# Set Object Enabled

**Menu:** `Actions/Object/Properties`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Sim`

## Funcao

Atualiza `Object Enabled` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Enabled` | `BOOLEAN` | `true` | - |

## Saidas

Nenhuma.
