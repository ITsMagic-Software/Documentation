# Set Allow Animations

**Menu:** `Actions/Object/Properties`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Sim`

## Funcao

Atualiza `Allow Animations` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Allow Animations` | `BOOLEAN` | `true` | - |

## Saidas

Nenhuma.
