# Swap Children Position

**Menu:** `Actions/Object/Hierarchy`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Sim`

## Funcao

Troca a ordem ou a posicao dos itens envolvidos em `Swap Children Position`.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Object A` | `GAME_OBJECT` | `Owner` | - |
| `Object B` | `GAME_OBJECT` | `Owner` | - |

## Saidas

Nenhuma.
