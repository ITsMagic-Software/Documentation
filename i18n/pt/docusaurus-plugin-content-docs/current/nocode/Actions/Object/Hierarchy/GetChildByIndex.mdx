# Get Child By Index

**Menu:** `Actions/Object/Hierarchy`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Nao`

## Funcao

Le um valor relacionado a `Child By Index` e disponibiliza o resultado nas saidas do node.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Index` | `NUMBER` | `0` | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Child` | `GAME_OBJECT` | - | - |
