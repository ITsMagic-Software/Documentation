# Find Child Object

**Menu:** `Actions/Object/Hierarchy`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Nao`

## Funcao

Procura o elemento ou recurso relacionado a `Child Object`.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Name` | `TEXT` | - | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Child` | `GAME_OBJECT` | - | - |
