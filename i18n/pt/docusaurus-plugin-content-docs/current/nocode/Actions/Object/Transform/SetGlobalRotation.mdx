# Set Global Rotation

**Menu:** `Actions/Object/Transform`

## Funcao

Atualiza `Global Rotation` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Global Rotation` | `QUATERNION` | `float4(0, 0, 0, 1)` | - |

## Saidas

Nenhuma.
