# Set Global Position

**Menu:** `Actions/Object/Transform`

## Funcao

Atualiza `Global Position` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Global Position` | `FLOAT3` | `float3(0, 0, 0)` | - |

## Saidas

Nenhuma.
