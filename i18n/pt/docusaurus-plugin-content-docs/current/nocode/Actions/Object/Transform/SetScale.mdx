# Set Scale

**Menu:** `Actions/Object/Transform`

## Funcao

Atualiza `Scale` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Scale` | `FLOAT3` | `float3(1, 1, 1)` | - |

## Saidas

Nenhuma.
