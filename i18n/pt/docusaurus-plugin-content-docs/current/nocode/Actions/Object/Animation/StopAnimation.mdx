# Stop Animation

**Menu:** `Actions/Object/Animation`

## Funcao

Executa a acao `Stop Animation` dentro do fluxo NoCode.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Object` | `GAME_OBJECT` | `Owner` | - |
| `Animation Name` | `TEXT` | - | - |
| `Transition Delay` | `NUMBER` | `0.5` | - |

## Saidas

Nenhuma.
