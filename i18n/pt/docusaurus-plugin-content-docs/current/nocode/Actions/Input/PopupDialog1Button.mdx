# Popup Dialog 1 Button

**Menu:** `Actions/Input`

## Funcao

Exibe um `JAVARuntime.PopupDialog` com um botao de confirmacao e trava o flow ate esse botao ser clicado.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Title` | `TEXT` | `Popup` | Titulo do dialog. |
| `Message` | `TEXT` | - | Mensagem do dialog. |
| `Type` | `POPUP_DIALOG_TYPE` | `SUCCESS` | Usa o enum mapeado para os estilos do `PopupDialog`. |
| `Confirm Button` | `TEXT` | `ok` | Rotulo usado no botao de confirmacao. |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `On Confirm` | `BRANCH` | - | Disparado quando o botao de confirmacao e clicado. |
