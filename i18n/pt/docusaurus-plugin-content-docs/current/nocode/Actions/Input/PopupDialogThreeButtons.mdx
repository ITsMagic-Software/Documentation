# Popup Dialog 3 Buttons

**Menu:** `Actions/Input`

## Funcao

Executa a acao `Popup Dialog 3 Buttons` dentro do fluxo NoCode.

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Title` | `TEXT` | - | - |
| `Message` | `TEXT` | - | - |
| `Type` | `POPUP_DIALOG_TYPE` | - | - |
| `Cancel Button` | `TEXT` | - | - |
| `Confirm Button` | `TEXT` | - | - |
| `Mid Button` | `TEXT` | - | - |
