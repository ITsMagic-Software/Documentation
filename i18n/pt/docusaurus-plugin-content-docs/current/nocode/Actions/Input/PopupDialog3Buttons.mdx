# Popup Dialog 3 Buttons

**Menu:** `Actions/Input`

## Funcao

Exibe um `JAVARuntime.PopupDialog` com botoes de cancelamento, confirmacao e mid e trava o flow ate um deles ser clicado.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Title` | `TEXT` | `Popup` | Titulo do dialog. |
| `Message` | `TEXT` | - | Mensagem do dialog. |
| `Type` | `POPUP_DIALOG_TYPE` | `SUCCESS` | Usa o enum mapeado para os estilos do `PopupDialog`. |
| `Cancel Button` | `TEXT` | `cancel` | Rotulo usado no botao de cancelamento. |
| `Confirm Button` | `TEXT` | `ok` | Rotulo usado no botao de confirmacao. |
| `Mid Button` | `TEXT` | `mid` | Rotulo usado no botao mid. |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `On Cancel` | `BRANCH` | - | Disparado quando o botao de cancelamento e clicado. |
| `On Confirm` | `BRANCH` | - | Disparado quando o botao de confirmacao e clicado. |
| `On Mid` | `BRANCH` | - | Disparado quando o botao mid e clicado. |
