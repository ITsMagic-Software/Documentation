# Is Key Down

**Menu:** `Actions/Buttons`

## Funcao

Verifica a condicao `Key Down` e encaminha o fluxo conforme o resultado.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Key Name` | `TEXT` | - | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `True` | `BRANCH` | - | - |
| `False` | `BRANCH` | - | - |
