# Is GamePad Button Pressed

**Menu:** `Actions/Buttons`

## Funcao

Verifica a condicao `GamePad Button Pressed` e encaminha o fluxo conforme o resultado.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Button Name` | `TEXT` | - | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `True` | `BRANCH` | - | - |
| `False` | `BRANCH` | - | - |
