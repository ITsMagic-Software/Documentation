# Is Key Pressed

**Menu:** `Actions/Buttons`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Nao`

## Funcao

Verifica a condicao `Key Pressed` e encaminha o fluxo conforme o resultado.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Key Name` | `TEXT` | - | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `True` | `BRANCH` | - | - |
| `False` | `BRANCH` | - | - |
