# Export Text To File

**Menu:** `Files`

## Propósito

Exporta um texto para o caminho de arquivo informado na entrada `File Path`.

## Entradas

| Nome | Tipo | Padrão | Detalhes |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | Conteúdo de texto a ser escrito. |
| `File Path` | `TEXT` | - | Caminho do arquivo de destino. |

## Saídas

Nenhuma.
