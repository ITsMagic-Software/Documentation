# Load Text From File (Project File)

**Menu:** `Files`

## Propósito

Carrega texto de um `ProjectFile` selecionado e retorna o texto resultante.

## Entradas

| Nome | Tipo | Padrão | Detalhes |
| --- | --- | --- | --- |
| `Project File` | `PROJECT_FILE` | - | Aceita um valor `ProjectFile`. |

## Saídas

| Nome | Tipo | Padrão | Detalhes |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | Texto carregado. |
