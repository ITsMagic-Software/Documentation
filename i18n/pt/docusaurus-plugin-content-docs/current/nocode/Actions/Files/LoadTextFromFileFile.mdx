# Load Text From File (File)

**Menu:** `Files`

## Propósito

Carrega texto de um `File` selecionado e retorna o texto resultante.

## Entradas

| Nome | Tipo | Padrão | Detalhes |
| --- | --- | --- | --- |
| `File` | `FILE` | - | Aceita um valor `File`. |

## Saídas

| Nome | Tipo | Padrão | Detalhes |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | Texto carregado. |
