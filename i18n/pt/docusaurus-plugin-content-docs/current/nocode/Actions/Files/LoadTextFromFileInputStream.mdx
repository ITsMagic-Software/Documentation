# Load Text From File (InputStream)

**Menu:** `Files`

## Propósito

Carrega texto de um `InputStream` e retorna o texto resultante.

## Entradas

| Nome | Tipo | Padrão | Detalhes |
| --- | --- | --- | --- |
| `Input Stream` | `INPUT_STREAM` | - | Aceita um valor `InputStream`. |

## Saídas

| Nome | Tipo | Padrão | Detalhes |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | Texto carregado. |
