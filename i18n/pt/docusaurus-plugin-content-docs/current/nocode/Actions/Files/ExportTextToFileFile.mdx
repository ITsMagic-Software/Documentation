# Export Text To File (File)

**Menu:** `Files`

## Propósito

Exporta um texto para o `File` selecionado.

## Entradas

| Nome | Tipo | Padrão | Detalhes |
| --- | --- | --- | --- |
| `Text` | `TEXT` | - | Conteúdo de texto a ser escrito. |
| `File` | `FILE` | - | Aceita um valor `File`. |

## Saídas

Nenhuma.
