# Set Material Color

**Menu:** `Actions/Material/Colors`

## Funcao

Atualiza `Material Color` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Material` | `MATERIAL` | `Owner` | - |
| `Color` | `COLOR` | `float4(1, 1, 1, 1)` | - |

## Saidas

Nenhuma.
