# Set Material Albedo

**Menu:** `Actions/Material/Textures`

## Funcao

Atualiza `Material Albedo` usando os valores recebidos nas entradas.

Se a entrada de textura usa um arquivo default, ele e carregado durante o loading async do mundo antes do inicio do gameplay.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Material` | `MATERIAL` | - | - |

## Saidas

Nenhuma.
