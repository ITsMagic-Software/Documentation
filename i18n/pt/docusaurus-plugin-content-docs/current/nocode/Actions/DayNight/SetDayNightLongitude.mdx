# Set Day Night Longitude

**Menu:** `Actions/DayNight`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Sim`

## Funcao

Atualiza `Day Night Longitude` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Longitude` | `NUMBER` | `0` | - |

## Saidas

Nenhuma.
