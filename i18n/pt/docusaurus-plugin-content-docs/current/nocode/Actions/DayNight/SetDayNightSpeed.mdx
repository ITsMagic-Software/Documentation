# Set Day Night Speed

**Menu:** `Actions/DayNight`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Sim`

## Funcao

Atualiza `Day Night Speed` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Speed` | `NUMBER` | `1` | - |

## Saidas

Nenhuma.
