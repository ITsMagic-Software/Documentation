# Set Time Of Day

**Menu:** `Actions/DayNight`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Sim`

## Funcao

Atualiza `Time Of Day` usando os valores recebidos nas entradas.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Hour` | `NUMBER` | `12` | - |

## Saidas

Nenhuma.
