# Is Loading

**Menu:** `Actions/World`

**Fluxo de entrada superior:** `Sim`

**Fluxo de saida superior:** `Nao`

## Funcao

Verifica a condicao `Loading` e encaminha o fluxo conforme o resultado.

## Entradas

Nenhuma.

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | - |
| `On False` | `BRANCH` | - | - |
