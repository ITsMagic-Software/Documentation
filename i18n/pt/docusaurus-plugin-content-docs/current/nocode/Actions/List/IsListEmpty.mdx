# Is Empty

**Menu:** `Actions/List`

## Funcao

Verifica a condicao `Empty` e encaminha o fluxo conforme o resultado.

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `List` | `LIST` | - | - |

## Saidas

| `On False` | `BRANCH` | - | - |
