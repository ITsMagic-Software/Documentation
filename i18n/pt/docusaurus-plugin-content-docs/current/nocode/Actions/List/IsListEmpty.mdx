# Is Empty

**Menu:** `Actions/List`

## Funcao

Verifica se a lista de entrada esta vazia.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `List` | `LIST` | - | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `On True` | `BRANCH` | - | Dispara quando a lista estiver vazia. |
| `On False` | `BRANCH` | - | Dispara quando a lista nao estiver vazia. |
