# Get Element At

**Menu:** `Actions/List`

## Funcao

Le um valor relacionado a `Element At` e disponibiliza o resultado nas saidas do node.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `List` | `LIST` | - | - |
| `Index` | `NUMBER` | - | - |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Element` | `DYNAMIC` | - | - |
