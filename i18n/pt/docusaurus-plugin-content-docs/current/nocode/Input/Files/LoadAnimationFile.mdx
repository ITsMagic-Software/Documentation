# Load Animation File

**Menu:** `Input/Files`

## Funcao

Carrega uma animacao a partir de um `AnimationFile` e retorna a instancia de animacao resultante.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Animation File` | `ANIMATION_FILE` | - | Aceita um valor `AnimationFile`. |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Animation` | `ANIMATION` | - | Instancia de animacao carregada. |
