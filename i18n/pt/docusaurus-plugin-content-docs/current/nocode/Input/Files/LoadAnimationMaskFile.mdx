# Load Animation Mask File

**Menu:** `Input/Files`

## Funcao

Carrega uma animation mask a partir de um `AnimationMaskFile` e retorna a instancia resultante.

## Entradas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Animation Mask File` | `ANIMATION_MASK_FILE` | - | Aceita um valor `AnimationMaskFile`. |

## Saidas

| Nome | Tipo | Padrao | Detalhes |
| --- | --- | --- | --- |
| `Animation Mask` | `ANIMATION_MASK` | - | Instancia carregada da animation mask. |
