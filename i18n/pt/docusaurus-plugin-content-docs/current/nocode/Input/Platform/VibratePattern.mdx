# Vibrate Pattern

**Menu:** `Input/Platform`

## Função

Vibra o dispositivo seguindo um padrão de tempos.

## Entradas

| Nome | Tipo | Padrão | Detalhes |
| --- | --- | --- | --- |
| `Timings` | `LIST` | - | Lista de tempos. |
| `Repeat` | `NUMBER` | - | Índice de repetição. |

## Saídas

Nenhuma.
